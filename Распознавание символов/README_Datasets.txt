Работу выполнил студент 325-й группы АЯ ВМК МГУ Зверев Дмитрий.

В архиве "Mini_OCR_DATASETS" находятся:
	- программа Start_Dataset.py для создания стартового датасета;
	- полученная после моего выполнения программы Start_Dataset.py папка 
	"start_dataset_original" со стартовым датасетом;
	- программа Modified_Dataset.py для создания расширенного аугментированного 
	датасета по стартовому;
	- полученная после моего выполнения программы Modified_Dataset.py папка
	"modified_dataset_original" с итоговым расширенным датасетом с аугментацией;
	- данный текстовый файл README.txt с описанием функционала программ по созданию
	датасетов и дополнительными пояснениями.

Для работоспособности программ датасетов и нейросети, находящейся в архиве на форуме
"Программа - распознавание символов" в moodle, необходимо распаковать оба архива в 
одну папку.

Датасет создавался на основе случайного выбора установленных на моем компьютере 
шрифтов формата .ttf при помощи программы Start_Dataset.py. Данная программа
создает папку "start_dataset", состоящую из 26 папок для каждой буквы по 100
образцов-изображений с заглавной буквой латинского алфавита.
!!! Однако при последующей проверке полученного стартового датасета можно
обнаружить, что некоторые образцы-изображения хранят некорректные символы
(прямоугольник, белый фон и т.п.). Программным образом устранить данную проблему не
удалось, поэтому некорректные изображения следует удалить ВРУЧНУЮ. Как правило, их
не больше 10% от общего числа изображений для каждой буквы, поэтому после ручного
удаления для каждой буквы останется по 90 образцов. !!!
Итак, стартовый датасет получен.

Теперь получим расширенный датасет с аугментацией. Для этого нужно запустить
программу Modified_Dataset.py, которая создаст папку "modified_dataset", так же
состоящую из 26 папок для хранения образцов-изображений с каждой буквой. Каждая
такая папка хранит по 90 изображений из стартового датасета, а также 270
аугментированных изображений, созданных на основе исходных 90, путем добавления
различных эффектов, таких как: шумы, царапины, инверсия, легкое размытие, 
наклоны/повороты и т.д.
Расширенный датасет получен.
