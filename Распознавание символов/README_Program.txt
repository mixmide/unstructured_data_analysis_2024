Работу выполнил студент 325-й группы АЯ ВМК МГУ Зверев Дмитрий.

В данном архиве находятся:
	- программа "Letter_Recognition.py" по распознаванию латинской буквы по фото;
	- папка с тестовой выборкой "alphabet_dataset_test", в которой хранится
	26 папок по 8 образцов-изображений для каждой буквы;
	- отчет о проделанной работе "Mini-OCR, Отчет.pdf";
	- данный текстовый файл README.txt с пояснениями к архиву.

Для работоспособности программы данный архив и архив с датасетами следует распаковать
в одну папку. Текущая версия программы настроена на последнюю модификацию (см. отчет),
время обучения - приблизительно 3 минуты. Комментарии к программе также присутствуют.

Эффективность построенной нейросети - 92%.
